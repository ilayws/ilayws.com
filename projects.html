Projects

Light Curve Inversion

A Python and C project aiming to reconstruct and identify 3D asteroid shapes only from their lightcurve graphs. Implements a simple raytracing algorithm and a DFF neural network.

Material Resonance Measurer

A Python program used to automate the collection of amplitude/frequency data and the resonance frequency of a sonic device using a laser Doppler vibrometer and a signal generator.

Wave Simulation in Non-Uniform Medium

Simulates a 1D wave (on a string) using a discrete expression of the wave equation. The wave velocity along the string can vary to simulate a non-uniform medium. Analyzed using FFT and Hilbert transform for instantaneous frequencies.

Cellular Life Simulator

A simulator combining ideas from different cellular automata models such as Conway's Game of Life and Lenia. (Link not up yet)

Exploit (Game)

A short 2D platformer game about exploiting the mechanic where leaving the screen from one end brings you back from the other end. Built in Godot.

8080 - Automatic Drum Machine

An automatic acoustic drumset with multiple drums and dynamically changing BPM to match a musician. Built using Arduino.

Firefly Synchronization

A Python model to study the factors affecting the synchronization of fireflies.

Polynomial and Bezier Interpolation

A Python project that takes n points as input and finds both the Bezier curve and the polynomial function for those points. This method works best for n < 11.
