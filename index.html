Polynomial and Bezier Interpolation

A python project that takes n points as input and finds the bezier curve and the polynomial function for those n points. The polynomial is found by creating a system of equations, expressing them in matrix form, reducing to echilon form with gaussian elimination, then calculating the polynomial's coefficients. This method works very well for n < 11 See more.
